## Команды

#### Запуск проекта

```
npm run start
```

#### Запуск в режиме разработки

```
npm run dev
```

#### Запуск сервера

```
npm run server
```

#### Сборка проекта

```
npm run build
```

#### Удаление собранных файлов

```
npm run clean
```

#### Запуск анализатора кода

```
npm run eslint
```

#### Запуск анализатора стиля

```
npm run stylelint
```
